## Featured Projects

### AI Avian Collision Avoidance System (AviSafe)

Real-time bird detection and deterrence system designed to protect endangered species at wind farms.

- Implemented YOLOv8 on Raspberry Pi 5 with Hailo AI HAT+ (26 TOPS)
- Processing 360-degree camera feeds at >400 FPS with offline-first architecture
- Cost-effective solution (INR 12,950 per turbine) targeting >72% collision reduction
- Addresses India's wind-bird crisis with annual mortality of 4,464 deaths per 1,000 km²

---

### IoT Cloudburst Early Warning System (MEGHADUT)

Distributed ESP32 sensor network with machine learning-powered rainfall classification for early flood warning.

- Tri-modal sensor fusion: Optical, piezo-acoustic, and tipping-bucket rainfall channels
- Custom acoustic drop-size distribution engine calibrated on 47,627 reference samples
- TinyML rainfall classifier achieving 96.4% macro F1 score with 4.3ms inference on ESP32
- Byzantine fault-tolerant consensus mechanism reducing false alarms by 96%
- Validated through 10,000-event Monte Carlo simulation

---

### EV Smart Support Platform (ChargeWise)

K! HACKS 3.0 Winner - Hybrid IoT and cloud platform for ensuring EV charging station reliability.

- Dual-verification engine combining ESP32 IoT sensors (70% weight) with crowdsourced user reports (30% weight)
- Real-time IoT telemetry pipeline syncing sensor data every 5 minutes
- 95% station status accuracy across 1,500+ charging points in Tamil Nadu
- Built and deployed in 24 hours during hackathon competition
- Eliminated wasted trips through intelligent routing and real-time status updates

---

### Hand Gesture Recognition and Distance Estimation

Real-time touchless human-computer interaction system using computer vision.

- Real-time gesture recognition using OpenCV and MediaPipe
- Tracking 21 hand landmarks at 43 FPS on CPU
- 96.8% top-1 classification accuracy across 2,200 test samples
- Stereo-geometry-based distance estimation with mean absolute error of 1.7 cm (20-80 cm operating range)
- Simultaneous gesture and depth inference in single pipeline
